## 1. 题目和例子

- 题目：给定两个字符串 `text1` 和 `text2`，求它们的最长公共子序列长度。
    
- **子序列（Subsequence）**：不要求连续，但相对顺序不能变。
    
- 示例 1  
    输入：`text1 = "abcde", text2 = "ace"`  
    输出：`3`
    
- 示例 2  
    输入：`text1 = "abc", text2 = "abc"`  
    输出：`3`
    
- 示例 3  
    输入：`text1 = "abc", text2 = "def"`  
    输出：`0`
    
- 再看一个小例子  
    输入：`text1 = "abac", text2 = "cab"`  
    输出：`2`  
    解释：最长公共子序列可以是 `"ab"`
    

---

## 2. 核心思路

- 这题是经典 **DP（Dynamic Programming，动态规划）**。
    
- 定义 `dp[i][j]`：表示 `text1` 前 `i` 个字符 和 `text2` 前 `j` 个字符 的最长公共子序列长度。
    

### 状态转移

如果当前两个字符相等：

```text
dp[i][j] = dp[i-1][j-1] + 1
```

如果不相等：

```text
dp[i][j] = Math.max(dp[i-1][j], dp[i][j-1])
```

### 为什么这样做

- 若 `text1[i-1] == text2[j-1]`  
    说明这个字符可以加入公共子序列，长度就在左上角基础上 `+1`
    
- 若不相等  
    说明这两个当前位置不能同时要，只能：
    
    - 丢掉 `text1` 当前字符，看 `dp[i-1][j]`
        
    - 或丢掉 `text2` 当前字符，看 `dp[i][j-1]`  
        取较大值
        

### 简短例子

`text1 = "abcde", text2 = "ace"`

- `a` 和 `a` 相等，贡献 `1`
    
- `c` 和 `c` 相等，再 `+1`
    
- `e` 和 `e` 相等，再 `+1`
    
- 最后答案是 `3`
    

### baseline 和 optimal

- baseline：二维 `dp`
    
- optimal：一维 `dp` 压缩空间
    
- 这题先把二维 `dp` 写熟最重要

## 3. ACM 题解

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String text1 = sc.nextLine();
        String text2 = sc.nextLine();

        int m = text1.length();
        int n = text2.length();

        int[][] dp = new int[m + 1][n + 1];

        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                } else {
                    dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
                }
            }
        }

        System.out.println(dp[m][n]);
    }
}
```

### ACM 输入示例

```text
abcde
ace
```

## 4. LeetCode 提交版

### 写法1：二维 DP

```java
class Solution {
    public int longestCommonSubsequence(String text1, String text2) {
        int m = text1.length();
        int n = text2.length();

        int[][] dp = new int[m + 1][n + 1];

        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
                    dp[i][j] = dp[i - 1][j - 1] + 1;
                } else {
                    dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
                }
            }
        }

        return dp[m][n];
    }
}
```

### 写法2：一维 DP

```java
class Solution {
    public int longestCommonSubsequence(String text1, String text2) {
        int m = text1.length();
        int n = text2.length();

        int[] dp = new int[n + 1];

        for (int i = 1; i <= m; i++) {
            int leftTop = 0; // 记录上一行上一列
            for (int j = 1; j <= n; j++) {
                int temp = dp[j];
                if (text1.charAt(i - 1) == text2.charAt(j - 1)) {
                    dp[j] = leftTop + 1;
                } else {
                    dp[j] = Math.max(dp[j], dp[j - 1]);
                }
                leftTop = temp;
            }
        }

        return dp[n];
    }
}
```

## 5. 一句话口诀

- **相等看左上加一，不等看上左最大。**
    

---

## 6. 易错点

- `dp[i][j]` 对应的是前 `i` 个、前 `j` 个字符，所以字符下标要写成 `i - 1`、`j - 1`
    
- 这是**子序列**，不是子串，不要求连续
    
- 不相等时不能写成左上角，应该取 `上` 和 `左` 的最大值

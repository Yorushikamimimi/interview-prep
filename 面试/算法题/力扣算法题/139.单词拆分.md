## 1. 题目和例子

- 题目：给你一个字符串 `s` 和一个单词字典 `wordDict`，判断 `s` 能不能被拆成一个或多个字典中的单词。
    
- 示例 1  
    输入：`s = "leetcode", wordDict = ["leet","code"]`  
    输出：`true`
    
- 示例 2  
    输入：`s = "applepenapple", wordDict = ["apple","pen"]`  
    输出：`true`
    
- 示例 3  
    输入：`s = "catsandog", wordDict = ["cats","dog","sand","and","cat"]`  
    输出：`false`
    
- 再看一个小例子  
    输入：`s = "aaaaaaa", wordDict = ["aaaa","aaa"]`  
    输出：`true`  
    解释：`"aaaa" + "aaa"`
    

---

## 2. 核心思路

- 这题是经典 **DP（Dynamic Programming，动态规划）**。
    
- 定义 `dp[i]`：表示前 `i` 个字符 `s[0...i-1]` 能不能被字典拆出来。
    

### 状态转移

如果存在某个 `j < i`，满足：

- `dp[j] == true`
    
- `s[j...i-1]` 在字典里
    

那么：

```text
dp[i] = true
```

### 为什么这样做

- 想知道前 `i` 个字符能不能拆
    
- 只要找到一个切分点 `j`
    
- 让前半段 `s[0...j-1]` 能拆，后半段 `s[j...i-1]` 又刚好在字典里
    
- 那前 `i` 个字符就能拆
    

### 初始化

- `dp[0] = true`
    
- 表示空串可以被成功拆分，作为起点
    

### 简短例子

`s = "leetcode"`，`wordDict = ["leet","code"]`

- `dp[0] = true`
    
- `"leet"` 在字典里，所以 `dp[4] = true`
    
- `"code"` 在字典里，且 `dp[4] = true`，所以 `dp[8] = true`
    
- 最后返回 `dp[8] = true`
    

### baseline 和 optimal

- baseline：纯递归枚举切分点，容易超时
    
- 主流正确解法：`dp + HashSet`
    
- `HashSet`：哈希集合，用来快速判断一个单词是否存在

## 3. ACM 题解

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        String s = sc.nextLine();
        int n = Integer.parseInt(sc.nextLine());

        List<String> wordDict = new ArrayList<>();
        for (int i = 0; i < n; i++) {
            wordDict.add(sc.nextLine());
        }

        Set<String> set = new HashSet<>(wordDict);
        boolean[] dp = new boolean[s.length() + 1];
        dp[0] = true;

        for (int i = 1; i <= s.length(); i++) {
            for (int j = 0; j < i; j++) {
                if (dp[j] && set.contains(s.substring(j, i))) {
                    dp[i] = true;
                    break;
                }
            }
        }

        System.out.println(dp[s.length()]);
    }
}
```

## 4. LeetCode 提交版

```java
import java.util.*;

class Solution {
    public boolean wordBreak(String s, List<String> wordDict) {
        Set<String> set = new HashSet<>(wordDict);
        boolean[] dp = new boolean[s.length() + 1];
        dp[0] = true;

        for (int i = 1; i <= s.length(); i++) {
            for (int j = 0; j < i; j++) {
                if (dp[j] && set.contains(s.substring(j, i))) {
                    dp[i] = true;
                    break;
                }
            }
        }

        return dp[s.length()];
    }
}
```

## 5. 一句话口诀

- **枚举切分点，前面能拆且后面在字典里。**
    

---

## 6. 易错点

- `dp[0] = true` 不能忘
    
- `dp[i]` 表示前 `i` 个字符，不是下标 `i`
    
- 判断单词是否存在时是 `s.substring(j, i)`，左闭右开别写错

## 1. 题目和例子

- 题目：给两个字符串 `word1` 和 `word2`，你可以对 `word1` 做三种操作：**插入**、**删除**、**替换**，求最少操作数把 `word1` 变成 `word2`。
    
- 示例 1  
    输入：`word1 = "horse", word2 = "ros"`  
    输出：`3`
    
- 示例 2  
    输入：`word1 = "intention", word2 = "execution"`  
    输出：`5`
    
- 再看一个小例子  
    输入：`word1 = "abc", word2 = "adc"`  
    输出：`1`  
    解释：把 `b` 替换成 `d`
    

---

## 2. 核心思路

- 这题是经典 **DP（Dynamic Programming，动态规划）**。
    
- 定义 `dp[i][j]`：表示 `word1` 前 `i` 个字符 变成 `word2` 前 `j` 个字符的最少操作数。
    

### 状态转移

如果当前字符相等：

```text
dp[i][j] = dp[i - 1][j - 1]
```

如果当前字符不相等，有三种选择：

```text
1. 删除：dp[i - 1][j] + 1
2. 插入：dp[i][j - 1] + 1
3. 替换：dp[i - 1][j - 1] + 1
```

所以：

```text
dp[i][j] = min(删除, 插入, 替换)
```

### 为什么这样做

假设看 `word1[i-1]` 和 `word2[j-1]`：

- 相等：最后一个字符不用动，直接看前面
    
- 不相等：
    
    - **删除**：删掉 `word1` 当前字符
        
    - **插入**：给 `word1` 末尾插一个和 `word2[j-1]` 一样的字符
        
    - **替换**：把 `word1` 当前字符改成 `word2[j-1]`
        

### 初始化

- `dp[i][0] = i`：把长度为 `i` 的串变空串，要删 `i` 次
    
- `dp[0][j] = j`：把空串变长度为 `j` 的串，要插 `j` 次
    

### 简短例子

`word1 = "ab", word2 = "ac"`

- `a` 和 `a` 相等，不用操作
    
- `b` 和 `c` 不等，替换一次即可
    
- 答案是 `1`
    

### baseline 和 optimal

- baseline：二维 `dp`
    
- optimal：一维 `dp` 压缩空间
    
- 先把二维版吃透最重要

## 3. ACM 题解

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String word1 = sc.nextLine();
        String word2 = sc.nextLine();

        int m = word1.length();
        int n = word2.length();

        int[][] dp = new int[m + 1][n + 1];

        // 初始化
        for (int i = 0; i <= m; i++) {
            dp[i][0] = i;
        }
        for (int j = 0; j <= n; j++) {
            dp[0][j] = j;
        }

        // 状态转移
        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (word1.charAt(i - 1) == word2.charAt(j - 1)) {
                    dp[i][j] = dp[i - 1][j - 1];
                } else {
                    int delete = dp[i - 1][j] + 1;
                    int insert = dp[i][j - 1] + 1;
                    int replace = dp[i - 1][j - 1] + 1;
                    dp[i][j] = Math.min(delete, Math.min(insert, replace));
                }
            }
        }

        System.out.println(dp[m][n]);
    }
}
```

## 4. LeetCode 提交版

### 写法1：二维 DP

```java
class Solution {
    public int minDistance(String word1, String word2) {
        int m = word1.length();
        int n = word2.length();

        int[][] dp = new int[m + 1][n + 1];

        for (int i = 0; i <= m; i++) {
            dp[i][0] = i;
        }
        for (int j = 0; j <= n; j++) {
            dp[0][j] = j;
        }

        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (word1.charAt(i - 1) == word2.charAt(j - 1)) {
                    dp[i][j] = dp[i - 1][j - 1];
                } else {
                    int delete = dp[i - 1][j] + 1;
                    int insert = dp[i][j - 1] + 1;
                    int replace = dp[i - 1][j - 1] + 1;
                    dp[i][j] = Math.min(delete, Math.min(insert, replace));
                }
            }
        }

        return dp[m][n];
    }
}
```

### 写法2：一维 DP

```java
class Solution {
    public int minDistance(String word1, String word2) {
        int m = word1.length();
        int n = word2.length();

        int[] dp = new int[n + 1];

        for (int j = 0; j <= n; j++) {
            dp[j] = j;
        }

        for (int i = 1; i <= m; i++) {
            int leftTop = dp[0];
            dp[0] = i;
            for (int j = 1; j <= n; j++) {
                int temp = dp[j];
                if (word1.charAt(i - 1) == word2.charAt(j - 1)) {
                    dp[j] = leftTop;
                } else {
                    int delete = temp + 1;
                    int insert = dp[j - 1] + 1;
                    int replace = leftTop + 1;
                    dp[j] = Math.min(delete, Math.min(insert, replace));
                }
                leftTop = temp;
            }
        }

        return dp[n];
    }
}
```

## 5. 一句话口诀

- **相等看左上，不等取删插改最小。**

## 6. 易错点

- `dp[i][0] = i`、`dp[0][j] = j` 别忘了初始化
    
- 插入、删除、替换对应的下标位置别搞混
    
- 这是把 `word1` 变成 `word2`，定义方向要始终一致

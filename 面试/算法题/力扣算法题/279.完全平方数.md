## 1. 题目和例子

- 题目：给你一个整数 `n`，求若干个**完全平方数**之和等于 `n` 时，最少需要几个数。
    
- **完全平方数（Perfect Square）**：例如 `1、4、9、16`，即某个整数平方得到的数。
    
- 示例 1  
    输入：`n = 12`  
    输出：`3`  
    解释：`12 = 4 + 4 + 4`
    
- 示例 2  
    输入：`n = 13`  
    输出：`2`  
    解释：`13 = 4 + 9`
    
- 再看两个小例子  
    输入：`n = 1`  
    输出：`1`
    
    输入：`n = 8`  
    输出：`2`  
    解释：`8 = 4 + 4`
    

---

## 2. 核心思路

- 这题主流解法是 **DP（Dynamic Programming，动态规划）**。
    
- 设 `dp[i]` 表示：凑出数字 `i` 所需要的最少完全平方数个数。
    

### 状态转移

对于每个 `i`，枚举所有不超过它的平方数 `j*j`：

```text
dp[i] = Math.min(dp[i], dp[i - j * j] + 1)
```

### 为什么这样做

- 如果最后选了一个平方数 `j*j`
    
- 那前面就还需要凑出 `i - j*j`
    
- 所以总个数就是：`dp[i - j*j] + 1`
    
- 枚举所有平方数，取最小值即可
    

### 初始化

- `dp[0] = 0`
    
- 其他位置先初始化成很大值
    

### 简短例子

`n = 12`

- `dp[1] = 1`，因为 `1`
    
- `dp[4] = 1`，因为 `4`
    
- `dp[8] = 2`，因为 `4 + 4`
    
- `dp[12] = 3`，因为 `4 + 4 + 4`
    

### baseline 和 optimal

- baseline：暴力递归，容易超时
    
- 主流正确解法：DP，时间复杂度 `O(n * sqrt(n))`
    
- 这题先把 DP 记住最稳

## 3. ACM 题解

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        int[] dp = new int[n + 1];
        Arrays.fill(dp, Integer.MAX_VALUE);

        dp[0] = 0;

        for (int i = 1; i <= n; i++) {
            for (int j = 1; j * j <= i; j++) {
                dp[i] = Math.min(dp[i], dp[i - j * j] + 1);
            }
        }

        System.out.println(dp[n]);
    }
}
```

## 4. LeetCode 提交版

```java
class Solution {
    public int numSquares(int n) {
        int[] dp = new int[n + 1];
        java.util.Arrays.fill(dp, Integer.MAX_VALUE);

        dp[0] = 0;

        for (int i = 1; i <= n; i++) {
            for (int j = 1; j * j <= i; j++) {
                dp[i] = Math.min(dp[i], dp[i - j * j] + 1);
            }
        }

        return dp[n];
    }
}
```

## 5. 一句话口诀

- **枚举最后一个平方数，取最小个数。**
    

---

## 6. 易错点

- `dp[0] = 0` 要有，表示凑出 0 不需要任何数
    
- 内层循环条件是 `j * j <= i`
    
- 这题求的是**最少个数**，所以用 `min`，不是求方案数

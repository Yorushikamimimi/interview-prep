## 1. 题目和例子

- 题目：给你一个只包含 `'('` 和 `')'` 的字符串 `s`，求最长**连续**有效括号子串的长度。
    
- 有效括号：左右括号能正确配对，比如 `"()"`、`"(())"`。
    

示例 1：

- 输入：`s = "(()"`
    
- 输出：`2`
    

示例 2：

- 输入：`s = ")()())"`
    
- 输出：`4`
    

示例 3：

- 输入：`s = ""`
    
- 输出：`0`
    

补充小例子：

- 输入：`s = "()(())"`
    
- 输出：`6`
    
- 因为整个串都有效

## 2. 核心思路

这题主流做法有两个：

- baseline：**栈 stack**
    
- optimal：其实面试里讲 **栈** 就完全够了，最直观、最好写
    

### 核心方法：栈存下标

栈里不直接存括号，存的是**下标**。

做法：

1. 先往栈里放一个 `-1`，表示“最后一个未匹配位置”
    
2. 遍历字符串：
    
    - 遇到 `'('`：把下标入栈
        
    - 遇到 `')'`：先弹栈
        
        - 如果弹完后栈空了，说明当前 `')'` 没法匹配，把当前下标入栈，作为新的分界点
            
        - 如果栈不空，当前有效长度就是 `i - stack.peek()`
            

### 为什么这样做

- 栈里始终保存“还没被匹配掉的位置”
    
- 当出现一段有效括号时，长度可以通过：
    
    - `当前下标 - 最近一个未匹配位置`  
        来计算
        

### 简短例子

`s = ")()())"`

过程简化：

- 初始栈：`[-1]`
    
- `)`：弹出后空，压入 `0`
    
- `(`：压入 `1`
    
- `)`：弹出 `1`，长度 = `2 - 0 = 2`
    
- `(`：压入 `3`
    
- `)`：弹出 `3`，长度 = `4 - 0 = 4`
    
- `)`：弹出后空，压入 `5`
    

最大值就是 `4`

## 3. ACM 题解

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        if (!sc.hasNextLine()) {
            System.out.println(0);
            return;
        }

        String s = sc.nextLine();
        Deque<Integer> stack = new ArrayDeque<>();
        stack.push(-1);

        int ans = 0;

        for (int i = 0; i < s.length(); i++) {
            char c = s.charAt(i);
            if (c == '(') {
                stack.push(i);
            } else {
                stack.pop();
                if (stack.isEmpty()) {
                    stack.push(i);
                } else {
                    ans = Math.max(ans, i - stack.peek());
                }
            }
        }

        System.out.println(ans);
    }
}
```

        Deque<Integer> stack = new ArrayDeque<>();
        stack.push(-1);

        int ans = 0;

        for (int i = 0; i < s.length(); i++) {
            char c = s.charAt(i);
            if (c == '(') {
                stack.push(i);
            } else {
                stack.pop();
                if (stack.isEmpty()) {
                    stack.push(i);

                }
            }
        }

    }
}

## 5. 一句话口诀

- **栈存下标，空了立界，长度相减。**
    

---

## 6. 易错点

1. 栈里存的是**下标**，不是字符。
    
2. 一开始要先压入 `-1`，这是关键边界。
    
3. 遇到 `')'` 弹栈后，如果空了，要把当前下标压回去作为新的分界点。

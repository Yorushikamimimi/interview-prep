## 1. 题目和例子

- 题目：给你一个字符串 `s`，返回其中最长的**回文子串**。
    
- **回文（Palindrome）**：正着读和反着读都一样的字符串。
    
- 示例 1  
    输入：`s = "babad"`  
    输出：`"bab"`  
    说明：`"aba"` 也对
    
- 示例 2  
    输入：`s = "cbbd"`  
    输出：`"bb"`
    
- 再看两个小例子  
    输入：`s = "a"`  
    输出：`"a"`
    
    输入：`s = "ac"`  
    输出：`"a"`  
    说明：`"c"` 也对
    

---

## 2. 核心思路

- 这题主流好写解法是 **中心扩展（Center Expansion，中心向两边扩散）**。
    
- 一个回文串一定有“中心”：
    
    - 奇数长度回文：中心是一个字符，如 `"aba"`
        
    - 偶数长度回文：中心是两个字符中间，如 `"abba"`
        

### 为什么这样做

- 枚举每个位置，把它当成回文中心，向左右两边扩展。
    
- 只要两边字符相等，就继续扩。
    
- 每次扩出来一个回文，更新最长答案。
    

### 为什么要分两种中心

- `"aba"` 这种是从一个点开始扩
    
- `"abba"` 这种是从两个点开始扩
    
- 所以每个位置都要做两次：
    
    - `expand(i, i)`
        
    - `expand(i, i + 1)`
        

### 简短例子

`s = "babad"`

- 以 `a` 为中心扩：得到 `"bab"` 或 `"aba"`
    
- 最长长度是 `3`
    

### baseline 和 optimal

- baseline：暴力枚举所有子串，再判断是不是回文，`O(n^3)`
    
- 主流正确解法：中心扩展，`O(n^2)`，面试很常用
    
- 还有 DP，但这题优先记中心扩展，更好写

## 3. ACM 题解

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        String s = sc.nextLine();

        if (s == null || s.length() == 0) {
            System.out.println("");
            return;
        }

        int start = 0;
        int maxLen = 1;

        for (int i = 0; i < s.length(); i++) {
            int len1 = expand(s, i, i);       // 奇数长度
            int len2 = expand(s, i, i + 1);   // 偶数长度
            int len = Math.max(len1, len2);

            if (len > maxLen) {
                maxLen = len;
                start = i - (len - 1) / 2;
            }
        }

        System.out.println(s.substring(start, start + maxLen));
    }

    public static int expand(String s, int left, int right) {
        while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)) {
            left--;
            right++;
        }
        return right - left - 1;
    }
}
```

## 4. LeetCode 提交版

```java
class Solution {
    public String longestPalindrome(String s) {
        if (s == null || s.length() == 0) {
            return "";
        }

        int start = 0;
        int maxLen = 1;

        for (int i = 0; i < s.length(); i++) {
            int len1 = expand(s, i, i);
            int len2 = expand(s, i, i + 1);
            int len = Math.max(len1, len2);

            if (len > maxLen) {
                maxLen = len;
                start = i - (len - 1) / 2;
            }
        }

        return s.substring(start, start + maxLen);
    }

    private int expand(String s, int left, int right) {
        while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)) {
            left--;
            right++;
        }
        return right - left - 1;
    }
}
```

## 5. 一句话口诀

- **每个点、每条缝都当中心，往两边扩。**

## 6. 易错点

- 既要处理奇数回文，也要处理偶数回文
    
- 扩展结束后，回文长度是 `right - left - 1`
    
- 起点更新公式别写错：`start = i - (len - 1) / 2`

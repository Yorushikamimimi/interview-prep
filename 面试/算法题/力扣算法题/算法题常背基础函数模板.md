
## 1）最大公约数 gcd

```java
public static int gcd(int a, int b) {
    while (b != 0) {
        int temp = a % b;
        a = b;
        b = temp;
    }
    return a;
}
```

含义：  
`gcd(a, b) = gcd(b, a % b)`，直到余数为 `0`。

---

## 2）最小公倍数 lcm

```java
public static int lcm(int a, int b) {
    return a / gcd(a, b) * b;
}
```

含义：  
`最小公倍数 = a * b / 最大公约数`  
先除后乘，避免溢出。

## 4）判断素数 prime

```java
public static boolean isPrime(int n) {
    if (n < 2) return false;
    for (int i = 2; i * i <= n; i++) {
        if (n % i == 0) return false;
    }
    return true;
}
```

含义：  
只需要判断到 `sqrt(n)`。

## 5）快速幂 qpow

```java
public static long qpow(long a, long b) {
    long res = 1;
    while (b > 0) {
        if ((b & 1) == 1) res *= a;
        a *= a;
        b >>= 1;
    }
    return res;
}
```

含义：  
**Fast Power（快速幂）**，用二进制拆指数。

## 7）前缀和 prefix sum

```java
int[] pre = new int[n + 1];
for (int i = 1; i <= n; i++) {
    pre[i] = pre[i - 1] + arr[i - 1];
}
```

求区间和 `[l, r]`：

```java
int sum = pre[r + 1] - pre[l];
```

含义：  
**Prefix Sum（前缀和）**，快速求区间和。

## 18）完整常用模板版

```java
import java.util.*;

public class Main {

    public static int gcd(int a, int b) {
        while (b != 0) {
            int temp = a % b;
            a = b;
            b = temp;
        }
        return a;
    }

    public static int lcm(int a, int b) {
        return a / gcd(a, b) * b;
    }

    public static boolean isPrime(int n) {
        if (n < 2) return false;
        for (int i = 2; i * i <= n; i++) {
            if (n % i == 0) return false;
        }
        return true;
    }

    public static long qpow(long a, long b) {
        long res = 1;
        while (b > 0) {
            if ((b & 1) == 1) res *= a;
            a *= a;
            b >>= 1;
        }
        return res;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int a = sc.nextInt();
        int b = sc.nextInt();

        System.out.println(gcd(a, b));
        System.out.println(lcm(a, b));
    }
}
```

## 1. 题目和例子

- 题目：给你一个数组 `nums`，每个位置表示一间房屋的钱。不能偷相邻两间，求一晚能偷到的最大金额。
    
- 示例 1  
    输入：`nums = [1,2,3,1]`  
    输出：`4`
    
- 示例 2  
    输入：`nums = [2,7,9,3,1]`  
    输出：`12`
    
- 再看一个小例子  
    输入：`nums = [2,1,1,2]`  
    输出：`4`  
    解释：偷第 1 间和第 4 间
    

---

## 2. 核心思路

- 这题是经典 **DP（Dynamic Programming，动态规划）**。
    
- 设 `dp[i]` 表示偷到第 `i` 间房时，能拿到的最大金额。
    

状态转移：

```text
dp[i] = Math.max(dp[i - 1], dp[i - 2] + nums[i])
```

### 为什么这样做

对于第 `i` 间房，只有两种选择：

- **不偷第 i 间**：那就是前一间的最优解 `dp[i - 1]`
    
- **偷第 i 间**：那前一间不能偷，只能接 `dp[i - 2] + nums[i]`
    

取两者最大值即可。

### 初始化

- `dp[0] = nums[0]`
    
- `dp[1] = Math.max(nums[0], nums[1])`
    

### 简短例子

`nums = [2,7,9,3,1]`

```text
dp[0] = 2
dp[1] = 7
dp[2] = max(7, 2+9) = 11
dp[3] = max(11, 7+3) = 11
dp[4] = max(11, 11+1) = 12
```

答案是 `12`

### baseline 和 optimal

- baseline：数组 `dp`
    
- optimal：只用两个变量滚动更新
    
- 这题和爬楼梯很像，但这里是“选或不选”

## 3. ACM 题解

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();
        int[] nums = new int[n];
        for (int i = 0; i < n; i++) {
            nums[i] = sc.nextInt();
        }

        if (n == 1) {
            System.out.println(nums[0]);
            return;
        }

        int prev2 = nums[0];
        int prev1 = Math.max(nums[0], nums[1]);

        for (int i = 2; i < n; i++) {
            int cur = Math.max(prev1, prev2 + nums[i]);
            prev2 = prev1;
            prev1 = cur;
        }

        System.out.println(prev1);
    }
}
```

### ACM 输入示例

```text
5
2 7 9 3 1
```

## 4. LeetCode 提交版

```java
class Solution {
    public int rob(int[] nums) {
        int n = nums.length;

        if (n == 1) {
            return nums[0];
        }

        int prev2 = nums[0];
        int prev1 = Math.max(nums[0], nums[1]);

        for (int i = 2; i < n; i++) {
            int cur = Math.max(prev1, prev2 + nums[i]);
            prev2 = prev1;
            prev1 = cur;
        }

        return prev1;
    }
}
```

## 5. 一句话口诀

- **偷当前就加前前，不偷当前就看前一。**

## 6. 易错点

- `dp[i]` 不是必须偷第 `i` 间，而是“到第 i 间为止的最优解”
    
- `n = 1` 要单独处理
    
- 转移是 `max(dp[i-1], dp[i-2] + nums[i])`，不是简单相加

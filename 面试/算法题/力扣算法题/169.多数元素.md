### 169. 多数元素

## 题目和例子

给定一个大小为 `n` 的数组 `nums` ，返回其中的多数元素。多数元素是指在数组中出现次数 **大于 `⌊ n / 2 ⌋`** 的元素。

你可以假设数组是非空的，并且给定的数组总是存在多数元素。

### 示例 1

**输入：**

```text
nums = [3,2,3]
```

**输出：**

```text
3
```

### 示例 2

**输入：**

```text
nums = [2,2,1,1,1,2,2]
```

**输出：**

```text
2
```

---

## 核心思路

最优解用 **Boyer-Moore Voting Algorithm（摩尔投票）**。

思路是“抵消”：

- 相同数字，票数 `+1`
    
- 不同数字，票数 `-1`
    
- 当票数变成 `0` 时，换候选人
    

因为多数元素个数 **大于一半**，所以其他所有元素一起也抵消不掉它，最后留下来的候选人就是答案。

---

## ACM 题解

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();
        int candidate = 0;
        int count = 0;

        for (int i = 0; i < n; i++) {
            int x = sc.nextInt();

            if (count == 0) {
                candidate = x;
            }

            if (x == candidate) {
                count++;
            } else {
                count--;
            }
        }

        System.out.println(candidate);
    }
}
```

## LeetCode 提交版

```java
class Solution {
    public int majorityElement(int[] nums) {
        int candidate = 0;
        int count = 0;

        for (int x : nums) {
            if (count == 0) {
                candidate = x;
            }

            if (x == candidate) {
                count++;
            } else {
                count--;
            }
        }

        return candidate;
    }
}
```

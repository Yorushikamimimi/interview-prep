## 1. 题目和例子

- 题目：爬到第 `n` 阶楼梯，每次只能爬 `1` 阶或 `2` 阶，求一共有多少种不同方法。
    
- 示例 1  
    输入：`n = 2`  
    输出：`2`
    
- 示例 2  
    输入：`n = 3`  
    输出：`3`
    
- 再看一个小例子  
    输入：`n = 4`  
    输出：`5`  
    解释：  
    `1+1+1+1`  
    `1+1+2`  
    `1+2+1`  
    `2+1+1`  
    `2+2`
    

---

## 2. 核心思路

- 这题是经典 **DP（Dynamic Programming，动态规划）**。
    
- 设 `dp[i]` 表示爬到第 `i` 阶的方法数。
    

状态转移：

```text
dp[i] = dp[i - 1] + dp[i - 2]
```

### 为什么这样做

- 到第 `i` 阶，最后一步只有两种可能：
    
    - 从第 `i-1` 阶爬 1 步上来
        
    - 从第 `i-2` 阶爬 2 步上来
        
- 所以总方法数就是这两种情况相加。
    

### 初始化

- `dp[1] = 1`
    
- `dp[2] = 2`
    

### 简短例子

`n = 5`

```text
dp[1]=1
dp[2]=2
dp[3]=3
dp[4]=5
dp[5]=8
```

### baseline 和 optimal

- baseline：数组 `dp`
    
- optimal：只用两个变量滚动更新
    
- 本质上就是 **Fibonacci（斐波那契数列）** 变形

## 3. ACM 题解

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int n = sc.nextInt();

        if (n == 1) {
            System.out.println(1);
            return;
        }
        if (n == 2) {
            System.out.println(2);
            return;
        }

        int a = 1;
        int b = 2;

        for (int i = 3; i <= n; i++) {
            int c = a + b;
            a = b;
            b = c;
        }

        System.out.println(b);
    }
}
```

## 4. LeetCode 提交版

```java
class Solution {
    public int climbStairs(int n) {
        if (n == 1) {
            return 1;
        }
        if (n == 2) {
            return 2;
        }

        int a = 1;
        int b = 2;

        for (int i = 3; i <= n; i++) {
            int c = a + b;
            a = b;
            b = c;
        }

        return b;
    }
}
```

## 5. 一句话口诀

- **到第 i 阶 = 前一阶方法数 + 前两阶方法数。**
    

---

## 6. 易错点

- `n=1` 和 `n=2` 要单独处理
    
- 这题是“方法数”，不是最小步数
    
- 状态转移是加法，不是取最大最小

## 1. 题目和例子

- 题目：给你一个 `m x n` 网格，机器人从左上角出发，每次只能**向右**或**向下**走一步，问到右下角一共有多少条不同路径。
    
- 示例 1  
    输入：`m = 3, n = 7`  
    输出：`28`
    
- 示例 2  
    输入：`m = 3, n = 2`  
    输出：`3`
    
- 可再理解一个小例子  
    输入：`m = 2, n = 2`  
    输出：`2`  
    解释：右下角只能通过  
    `右 -> 下` 或 `下 -> 右`
    

---

## 2. 核心思路

- 这题是经典 **DP（Dynamic Programming，动态规划）**。
    
- 设 `dp[i][j]` 表示走到 `i,j` 这个位置的路径数。
    
- 因为机器人只能从**上边**或者**左边**走过来，所以：
    

```text
dp[i][j] = dp[i-1][j] + dp[i][j-1]
```

### 为什么这样做

- 到当前格子的所有路径，只有两种来源：
    
    - 从上面下来
        
    - 从左边过来
        
- 所以当前路径数 = 上方路径数 + 左方路径数
    

### 初始化

- 第一行只能一直向右走，所以全是 `1`
    
- 第一列只能一直向下走，所以全是 `1`
    

### 简短例子

`m=3, n=3`

```text
1 1 1
1 2 3
1 3 6
```

最后答案就是右下角 `6`

### 补一句

- baseline：二维 `dp`
    
- optimal：一维 `dp` 压缩空间  
    这题先掌握二维，再记一维优化。

## 3. ACM 题解

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int m = sc.nextInt();
        int n = sc.nextInt();

        int[][] dp = new int[m][n];

        // 初始化第一列
        for (int i = 0; i < m; i++) {
            dp[i][0] = 1;
        }

        // 初始化第一行
        for (int j = 0; j < n; j++) {
            dp[0][j] = 1;
        }

        // 状态转移
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                dp[i][j] = dp[i - 1][j] + dp[i][j - 1];
            }
        }

        System.out.println(dp[m - 1][n - 1]);
    }
}
```

## 4. LeetCode 提交版

### 写法1：二维 DP

```java
class Solution {
    public int uniquePaths(int m, int n) {
        int[][] dp = new int[m][n];

        for (int i = 0; i < m; i++) {
            dp[i][0] = 1;
        }

        for (int j = 0; j < n; j++) {
            dp[0][j] = 1;
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                dp[i][j] = dp[i - 1][j] + dp[i][j - 1];
            }
        }

        return dp[m - 1][n - 1];
    }
}
```

### 写法2：一维 DP（更优）

```java
class Solution {
    public int uniquePaths(int m, int n) {
        int[] dp = new int[n];

        for (int j = 0; j < n; j++) {
            dp[j] = 1;
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                dp[j] = dp[j] + dp[j - 1];
            }
        }

        return dp[n - 1];
    }
}
```

## 6. 易错点

- 第一行和第一列要先初始化为 `1`
    
- `dp[i][j]` 不是取最大值，是 **上边 + 左边**
    
- 一维 `dp` 时，内层循环要从左到右更新

## 1. 题目和例子

- 题目：给你一个整数数组 `nums`，找出乘积最大的 **连续子数组**，返回这个最大乘积。
    
- 连续子数组：元素必须连续。
    

示例 1：

- 输入：`nums = [2,3,-2,4]`
    
- 输出：`6`
    

示例 2：

- 输入：`nums = [-2,0,-1]`
    
- 输出：`0`
    

补充小例子：

- 输入：`nums = [-2,3,-4]`
    
- 输出：`24`
    
- 解释：`(-2) * 3 * (-4) = 24`
    

---

## 2. 核心思路

这题不能只维护“最大值”，还要同时维护“最小值”。

原因：

- 当前数如果是负数，**最小值 × 负数** 可能变成最大值。
    
- 所以每到一个位置，都要知道：
    
    - 以当前元素结尾的最大乘积 `maxDp`
        
    - 以当前元素结尾的最小乘积 `minDp`
        

转移：

- 当前数是 `x`
    
- 新的最大值来自这三种：
    
    - `x`
        
    - `x * maxDp`
        
    - `x * minDp`
        
- 新的最小值也来自这三种里的最小者
    

### 简短例子

`nums = [-2,3,-4]`

过程：

- 开始：`maxDp = -2, minDp = -2, ans = -2`
    
- 到 `3`：
    
    - 最大是 `3`
        
    - 最小是 `-6`
        
- 到 `-4`：
    
    - `-6 * -4 = 24`，变成最大值
        
- 最终答案：`24`

## 3. ACM 题解

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        List<Integer> list = new ArrayList<>();

        while (sc.hasNextInt()) {
            list.add(sc.nextInt());
        }

        if (list.size() == 0) {
            System.out.println(0);
            return;
        }

        int maxDp = list.get(0);
        int minDp = list.get(0);
        int ans = list.get(0);

        for (int i = 1; i < list.size(); i++) {
            int x = list.get(i);
            int a = x;
            int b = x * maxDp;
            int c = x * minDp;

            int newMax = Math.max(a, Math.max(b, c));
            int newMin = Math.min(a, Math.min(b, c));

            maxDp = newMax;
            minDp = newMin;
            ans = Math.max(ans, maxDp);
        }

        System.out.println(ans);
    }
}
```

## 4. LeetCode 提交版

```java
class Solution {
    public int maxProduct(int[] nums) {
        int maxDp = nums[0];
        int minDp = nums[0];
        int ans = nums[0];

        for (int i = 1; i < nums.length; i++) {
            int x = nums[i];
            int a = x;
            int b = x * maxDp;
            int c = x * minDp;

            int newMax = Math.max(a, Math.max(b, c));
            int newMin = Math.min(a, Math.min(b, c));

            maxDp = newMax;
            minDp = newMin;
            ans = Math.max(ans, maxDp);
        }

        return ans;
    }
}
```

## 5. 一句话口诀

- **乘积看双值，最大最小一起存。**
    

---

## 6. 易错点

1. 不能只维护最大乘积，负数会让最小值翻身变最大。
    
2. 这里是 **连续子数组**，不是子序列。
    
3. 更新时要先用旧值算出 `newMax` 和 `newMin`，不要先改一个再算另一个。

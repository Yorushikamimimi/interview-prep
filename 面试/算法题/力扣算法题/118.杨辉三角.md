## 1. 题目和例子

- 题目：给定一个整数 `numRows`，生成杨辉三角的前 `numRows` 行。
    
- **杨辉三角（Pascal's Triangle）**：每个数等于它左上和右上的数之和。
    
- 示例 1  
    输入：`numRows = 5`  
    输出：`[[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]`
    
- 示例 2  
    输入：`numRows = 1`  
    输出：`[[1]]`
    
- 再看一个小例子  
    输入：`numRows = 4`  
    输出：`[[1],[1,1],[1,2,1],[1,3,3,1]]`
    

---

## 2. 核心思路

- 这题直接按定义模拟生成。
    
- 每一行的：
    
    - 第一个数一定是 `1`
        
    - 最后一个数一定是 `1`
        
    - 中间位置：`当前行[j] = 上一行[j - 1] + 上一行[j]`
        

### 为什么这样做

因为杨辉三角本身就是这个规律：

- 边界永远是 `1`
    
- 中间数由上一行两个位置相加得到
    

### 简短例子

第 4 行怎么来：

```text
上一行：1 2 1
当前行：1 3 3 1
        1 (1+2) (2+1) 1
```

### baseline 和 optimal

- baseline：直接用二维列表逐行生成
    
- 这题主流解法就是模拟，没有更复杂的必要

## 3. ACM 题解

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int numRows = sc.nextInt();

        List<List<Integer>> res = new ArrayList<>();

        for (int i = 0; i < numRows; i++) {
            List<Integer> row = new ArrayList<>();

            for (int j = 0; j <= i; j++) {
                if (j == 0 || j == i) {
                    row.add(1);
                } else {
                    row.add(res.get(i - 1).get(j - 1) + res.get(i - 1).get(j));
                }
            }

            res.add(row);
        }

        System.out.println(res);
    }
}
```

## 4. LeetCode 提交版

```java
import java.util.*;

class Solution {
    public List<List<Integer>> generate(int numRows) {
        List<List<Integer>> res = new ArrayList<>();

        for (int i = 0; i < numRows; i++) {
            List<Integer> row = new ArrayList<>();

            for (int j = 0; j <= i; j++) {
                if (j == 0 || j == i) {
                    row.add(1);
                } else {
                    row.add(res.get(i - 1).get(j - 1) + res.get(i - 1).get(j));
                }
            }

            res.add(row);
        }

        return res;
    }
}
```

## 5. 一句话口诀

- **两边都是 1，中间等于左上加右上。**

## 6. 易错点

- `j == 0` 和 `j == i` 时要直接放 `1`
    
- 中间位置取的是**上一行**的 `j-1` 和 `j`
    
- 循环层数别写错，当前第 `i` 行一共有 `i+1` 个数

## 1. 题目和例子

- 题目：给你一个只包含正整数的非空数组 `nums`，判断能否分成两个子集，使两个子集元素和相等。
    
- 本质：看能不能从数组里选出一些数，和恰好等于 `sum / 2`。
    

示例 1：

- 输入：`nums = [1,5,11,5]`
    
- 输出：`true`
    

示例 2：

- 输入：`nums = [1,2,3,5]`
    
- 输出：`false`
    

补充小例子：

- 输入：`nums = [1,2,5]`
    
- 输出：`false`
    
- 因为总和是 `8`，目标是 `4`，但凑不出 `4`
    

---

## 2. 核心思路

这是经典 **0/1 Knapsack**，0/1 背包：每个数只能选一次。

### 为什么这样做

如果数组总和是 `sum`：

- 如果 `sum` 是奇数，不可能平分，直接 `false`
    
- 如果 `sum` 是偶数，就看能不能选出若干个数，使和等于 `sum / 2`
    

于是问题变成：

- 是否存在一个子集，和为 `target = sum / 2`
    

### DP 定义

用一维布尔数组 `dp[j]`：

- `dp[j] = true` 表示：是否能从前面若干个数里选出和为 `j`
    

初始：

- `dp[0] = true`，因为什么都不选就能凑出 0
    

转移：

- 对每个数 `num`
    
- 从后往前枚举 `j`
    
- `dp[j] = dp[j] || dp[j - num]`
    

为什么倒序：

- 防止一个数被重复使用
    
- 倒序才能保证每个数只用一次
    

### 简短例子

`nums = [1,5,11,5]`

- 总和 `22`，目标 `11`
    
- 其实单独 `11` 就已经能凑出
    
- 所以答案是 `true`

## 3. ACM 题解

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        List<Integer> list = new ArrayList<>();

        while (sc.hasNextInt()) {
            list.add(sc.nextInt());
        }

        int n = list.size();
        if (n == 0) {
            System.out.println(false);
            return;
        }

        int sum = 0;
        for (int num : list) {
            sum += num;
        }

        if ((sum & 1) == 1) {
            System.out.println(false);
            return;
        }

        int target = sum / 2;
        boolean[] dp = new boolean[target + 1];
        dp[0] = true;

        for (int num : list) {
            for (int j = target; j >= num; j--) {
                dp[j] = dp[j] || dp[j - num];
            }
        }

        System.out.println(dp[target]);
    }
}
```

## 4. LeetCode 提交版

```java
class Solution {
    public boolean canPartition(int[] nums) {
        int sum = 0;
        for (int num : nums) {
            sum += num;
        }

        if ((sum & 1) == 1) {
            return false;
        }

        int target = sum / 2;
        boolean[] dp = new boolean[target + 1];
        dp[0] = true;

        for (int num : nums) {
            for (int j = target; j >= num; j--) {
                dp[j] = dp[j] || dp[j - num];
            }
        }

        return dp[target];
    }
}
```

## 5. 一句话口诀

- **先看总和奇偶，再做 0/1 背包。**
    

---

## 6. 易错点

1. 总和是奇数时，直接 `false`。
    
2. `j` 必须倒序遍历，否则一个数会被重复选。
    
3. 这题求的是“能不能凑出来”，不是“最多能凑多大”。

### 136. 只出现一次的数字

给你一个 **非空整数数组** `nums` ，除了某个元素只出现 **一次** 以外，其余每个元素均出现 **两次**。找出那个只出现了一次的元素。

你必须设计并实现 **线性时间复杂度** 的算法来解决此问题，且该算法只使用 **常量额外空间**。

---

### 示例 1

**输入：**

```text
nums = [2,2,1]
```

**输出：**

```text
1
```

---

### 示例 2

**输入：**

```text
nums = [4,1,2,1,2]
```

**输出：**

```text
4
```

---

### 示例 3

**输入：**

```text
nums = [1]
```

**输出：**

```text
1
```

### 核心思路

这题用 **XOR（异或）**。

因为：

```java
a ^ a = 0
a ^ 0 = a
```

数组里成对出现的数字，全部异或后都会抵消掉，最后剩下的就是只出现一次的数字。

时间复杂度：**O(n)**  
空间复杂度：**O(1)**

---

### ACM 题解（Java）

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();
        int ans = 0;

        for (int i = 0; i < n; i++) {
            ans ^= sc.nextInt();
        }

        System.out.println(ans);
    }
}
```

---

### LeetCode 提交版

```java
class Solution {
    public int singleNumber(int[] nums) {
        int ans = 0;
        for (int x : nums) {
            ans ^= x;
        }
        return ans;
    }
}
```

---

### 一句话口诀

**全部异或一遍，成对抵消，剩下答案。**

---

### 这题你要记住的点

1. **不用排序**
    
2. **不用 HashMap**
    
3. **最优解就是异或**
    
4. **异或看二进制位，不看数字位置**
    

再压缩成一句就是：

**相同为 0，不同留位；全体异或，单独留下。**

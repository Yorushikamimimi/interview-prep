## 1. 题目和例子

- 题目：给定一个 `m x n` 的非负整数网格 `grid`，从左上角走到右下角，每次只能向右或向下，求路径数字和的最小值。
    
- 示例 1  
    输入：`grid = [[1,3,1],[1,5,1],[4,2,1]]`  
    输出：`7`  
    解释：最小路径是 `1 -> 3 -> 1 -> 1 -> 1`
    
- 示例 2  
    输入：`grid = [[1,2,3],[4,5,6]]`  
    输出：`12`
    
- 再看一个小例子  
    输入：`grid = [[1,2],[1,1]]`  
    输出：`3`  
    解释：走法 `1 -> 1 -> 1` 更小
    

---

## 2. 核心思路

- 这题也是 **DP（Dynamic Programming，动态规划）**。
    
- 设 `dp[i][j]` 表示：从左上角走到 `i,j` 位置的**最小路径和**。
    

状态转移：

```text
dp[i][j] = min(dp[i-1][j], dp[i][j-1]) + grid[i][j]
```

### 为什么这样做

- 到当前格子，只能从：
    
    - 上边走下来
        
    - 左边走过来
        
- 那当然选这两条路里更小的，再加上当前格子的值。
    

### 初始化

- `dp[0][0] = grid[0][0]`
    
- 第一行只能一直从左边过来
    
- 第一列只能一直从上边下来
    

### 简短例子

`grid = [[1,3,1],[1,5,1],[4,2,1]]`

`dp` 过程大致是：

```text
1 4 5
2 7 6
6 8 7
```

右下角就是答案 `7`

### baseline 和 optimal

- baseline：二维 `dp`
    
- optimal：直接在原数组上改，或用一维 `dp`
    
- 面试先把二维写稳，再记空间优化

## 3. ACM 题解

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int m = sc.nextInt();
        int n = sc.nextInt();

        int[][] grid = new int[m][n];
        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                grid[i][j] = sc.nextInt();
            }
        }

        int[][] dp = new int[m][n];
        dp[0][0] = grid[0][0];

        // 初始化第一列
        for (int i = 1; i < m; i++) {
            dp[i][0] = dp[i - 1][0] + grid[i][0];
        }

        // 初始化第一行
        for (int j = 1; j < n; j++) {
            dp[0][j] = dp[0][j - 1] + grid[0][j];
        }

        // 状态转移
        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                dp[i][j] = Math.min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j];
            }
        }

        System.out.println(dp[m - 1][n - 1]);
    }
}
```

### ACM 输入示例

```text
3 3
1 3 1
1 5 1
4 2 1
```

## 4. LeetCode 提交版

### 写法1：二维 DP

```java
class Solution {
    public int minPathSum(int[][] grid) {
        int m = grid.length;
        int n = grid[0].length;

        int[][] dp = new int[m][n];
        dp[0][0] = grid[0][0];

        for (int i = 1; i < m; i++) {
            dp[i][0] = dp[i - 1][0] + grid[i][0];
        }

        for (int j = 1; j < n; j++) {
            dp[0][j] = dp[0][j - 1] + grid[0][j];
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                dp[i][j] = Math.min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j];
            }
        }

        return dp[m - 1][n - 1];
    }
}
```

### 写法2：一维 DP

```java
class Solution {
    public int minPathSum(int[][] grid) {
        int m = grid.length;
        int n = grid[0].length;

        int[] dp = new int[n];
        dp[0] = grid[0][0];

        for (int j = 1; j < n; j++) {
            dp[j] = dp[j - 1] + grid[0][j];
        }

        for (int i = 1; i < m; i++) {
            dp[0] = dp[0] + grid[i][0];
            for (int j = 1; j < n; j++) {
                dp[j] = Math.min(dp[j], dp[j - 1]) + grid[i][j];
            }
        }

        return dp[n - 1];
    }
}
```

- **到当前格，取上左最小再加自己。**

## 6. 易错点

- 第一行、第一列不是赋值 `1`，而是**累加**
    
- 这题求的是 **最小和**，不是路径条数
    
- 一维 `dp` 里，`dp[j]` 表示上方，`dp[j-1]` 表示左方，不要搞反

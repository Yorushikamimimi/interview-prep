## 题目和例子

给定一个包含 `n + 1` 个整数的数组 `nums` ，其数字都在 `[1, n]` 范围内（包括 `1` 和 `n`），可知至少存在一个重复的整数。

假设 `nums` 只有 **一个重复的整数** ，返回这个重复的数。

你设计的解决方案必须 **不修改数组** `nums` 且只用 **常量级 O(1)** 的额外空间。

### 示例 1

**输入：**

```text
nums = [1,3,4,2,2]
```

**输出：**

```text
2
```

### 示例 2

**输入：**

```text
nums = [3,1,3,4,2]
```

**输出：**

```text
3
```

### 示例 3

**输入：**

```text
nums = [3,3,3,3,3]
```

**输出：**

```text
3
```

---

## 核心思路

最优解用 **Floyd Cycle Detection（快慢指针判环）**。

把数组看成“链表”：

- 下标当作节点
    
- `nums[i]` 当作 next 指针
    

因为数组长度是 `n+1`，但数值只在 `1~n`，所以一定会形成环。  
而**环的入口**，就是那个重复数字。

### 两步：

1. 快慢指针先相遇
    
2. 一个指针回到起点，两指针一起一步一步走，再次相遇的位置就是重复数
    

---

## ACM 题解

```java
import java.util.*;

public class Main {
    public static int findDuplicate(int[] nums) {
        int slow = nums[0];
        int fast = nums[0];

        // 第一次相遇
        do {
            slow = nums[slow];
            fast = nums[nums[fast]];
        } while (slow != fast);

        // 找环入口
        int ptr = nums[0];
        while (ptr != slow) {
            ptr = nums[ptr];
            slow = nums[slow];
        }

        return ptr;
    }

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();   // 输入数组长度
        int[] nums = new int[n];

        for (int i = 0; i < n; i++) {
            nums[i] = sc.nextInt();
        }

        System.out.println(findDuplicate(nums));
    }
}
```

## LeetCode 提交版

```java
class Solution {
    public int findDuplicate(int[] nums) {
        int slow = nums[0];
        int fast = nums[0];

        do {
            slow = nums[slow];
            fast = nums[nums[fast]];
        } while (slow != fast);

        int ptr = nums[0];
        while (ptr != slow) {
            ptr = nums[ptr];
            slow = nums[slow];
        }

        return ptr;
    }
}
```

## 一句话口诀

**数组映射成链表，快慢指针找环口，环入口就是重复数。**

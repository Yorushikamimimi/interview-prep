## 1. 题目和例子

- 题目：给你一个硬币数组 `coins`，每种硬币可以用无限次，再给一个总金额 `amount`，求凑出该金额所需的最少硬币数；凑不出来返回 `-1`。
    
- 示例 1  
    输入：`coins = [1,2,5], amount = 11`  
    输出：`3`  
    解释：`11 = 5 + 5 + 1`
    
- 示例 2  
    输入：`coins = [2], amount = 3`  
    输出：`-1`
    
- 示例 3  
    输入：`coins = [1], amount = 0`  
    输出：`0`
    
- 再看一个小例子  
    输入：`coins = [2,5,10], amount = 6`  
    输出：`3`  
    解释：`6 = 2 + 2 + 2`
    

---

## 2. 核心思路

- 这题是经典 **DP（Dynamic Programming，动态规划）**，属于**完全背包**。
    
- **完全背包**：每个物品可以选无限次。
    
- 定义 `dp[i]`：凑出金额 `i` 所需的最少硬币数。
    

### 状态转移

对于每个硬币 `coin`，如果 `i >= coin`：

```text
dp[i] = Math.min(dp[i], dp[i - coin] + 1)
```

### 为什么这样做

- 如果最后一枚用的是 `coin`
    
- 那前面就要先凑出 `i - coin`
    
- 所以总硬币数 = `dp[i - coin] + 1`
    
- 枚举所有硬币，取最小值
    

### 初始化

- `dp[0] = 0`
    
- 其他位置先设成一个很大值，表示暂时凑不出来
    

### 简短例子

`coins = [1,2,5], amount = 11`

- `dp[1] = 1`
    
- `dp[2] = 1`
    
- `dp[5] = 1`
    
- `dp[10] = 2`
    
- `dp[11] = 3`
    

### baseline 和 optimal

- baseline：暴力递归，容易超时
    
- 主流正确解法：一维 DP
    
- 这题先记住 `dp[i] = min(dp[i], dp[i - coin] + 1)`

## 3. ACM 题解

```java
import java.util.*;

public class Main {
    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int n = sc.nextInt();      // 硬币种类数
        int amount = sc.nextInt(); // 目标金额

        int[] coins = new int[n];
        for (int i = 0; i < n; i++) {
            coins[i] = sc.nextInt();
        }

        int[] dp = new int[amount + 1];
        Arrays.fill(dp, amount + 1);

        dp[0] = 0;

        for (int coin : coins) {
            for (int i = coin; i <= amount; i++) {
                dp[i] = Math.min(dp[i], dp[i - coin] + 1);
            }
        }

        System.out.println(dp[amount] > amount ? -1 : dp[amount]);
    }
}
```

### ACM 输入示例

```text
3 11
1 2 5
```

## 4. LeetCode 提交版

```java
import java.util.*;

class Solution {
    public int coinChange(int[] coins, int amount) {
        int[] dp = new int[amount + 1];
        Arrays.fill(dp, amount + 1);

        dp[0] = 0;

        for (int coin : coins) {
            for (int i = coin; i <= amount; i++) {
                dp[i] = Math.min(dp[i], dp[i - coin] + 1);
            }
        }

        return dp[amount] > amount ? -1 : dp[amount];
    }
}
```

## 5. 一句话口诀

- **枚举硬币刷背包，当前最优看前面加一。**

## 6. 易错点

- `dp` 初始值不能乱设，常用 `amount + 1` 表示不可能
    
- `dp[0] = 0` 别忘
    
- 最后要判断是否真的能凑出，不能就返回 `-1`
